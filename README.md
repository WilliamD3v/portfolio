# Portfolio Programador - Next.js + Tailwind

Projeto de front-end completo para divulgação profissional como programador, com páginas de apresentação, serviços, portfólio, orçamento e agendamento.

## Tecnologias

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Lucide React

## Páginas incluídas

- Home
- Sobre
- Serviços
- Portfólio
- Orçamento com simulador
- Agendamento
- Contato

## Como rodar

```bash
npm install
npm run dev
```

A aplicação ficará disponível em:

```bash
http://localhost:3000
```

## Observações

- O projeto já está pronto para servir como base visual e estrutural.
- Os formulários de orçamento, agendamento e contato estão prontos no front-end.
- Para uso real, você pode integrar com WhatsApp, email, banco de dados, API própria ou Google Calendar.
- Troque os textos, contatos e cases pelos seus dados reais.

## Estrutura

```bash
src/
  app/
  components/
  data/
  lib/
  types/
```

## Personalizações sugeridas

- Adicionar sua foto ou identidade visual
- Substituir depoimentos fictícios por reais
- Integrar o agendamento com backend
- Salvar leads do orçamento em banco de dados
- Conectar formulário de contato com email ou WhatsApp
